Breakpoints

According to Oxford dictionary, a breakpoint is a place or time at which an interruption or change is made. In a CSS file we can use breakpoints to define how the layout of a webpage can change depending on the size of the screen or device you are using. For example using a breakpoint will prevent a paragraph or text to become distorted and make it unreadable when the screen gets bigger or smaller. The points are coded in the CSS file as pixel values and when a website reaches a certain value the layout will adapt itself to make the content more readable and user friendly.
